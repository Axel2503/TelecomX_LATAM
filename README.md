# 📊 Proyecto: Análisis de Evasión de Clientes (Churn) - Telecom X

## 🎯 Propósito del Análisis
Este proyecto tiene como objetivo analizar la base de datos de los clientes de la empresa de telecomunicaciones **"Telecom X"** para comprender a fondo el fenómeno de la evasión (conocido en la industria como *Churn*). 

A través de la limpieza de datos, la ingeniería de características y el análisis exploratorio visual usando Python, buscamos identificar qué factores (como el tipo de contrato, el tiempo de permanencia o los cargos mensuales) influyen más en la decisión de un cliente de cancelar su servicio. El propósito final es extraer *insights* valiosos que permitan proponer estrategias efectivas para la retención de usuarios.

## 📂 Estructura del Proyecto y Organización de Archivos

El proyecto está organizado de manera clara para facilitar su lectura y reproducción. Los archivos principales son los siguientes:

* 📄 **`TelecomX_Data.json`**: Archivo original descargado desde la API. Contiene los datos crudos de los clientes (demografía, servicios contratados, gastos, etc.).
* 📖 **`TelecomX_diccionario.md`**: Documento de referencia que explica detalladamente el significado de cada columna y variable dentro de la base de datos original.
* 📓 **`Analisis_TelecomX.ipynb`**: Cuaderno interactivo (Jupyter Notebook / Google Colab) que contiene el núcleo del proyecto. Aquí se documenta paso a paso:
  * La importación y limpieza de los datos.
  * La transformación de variables categóricas a numéricas.
  * La generación de visualizaciones (gráficos de barras, de pastel y diagramas de caja).
  * El informe final con las conclusiones y recomendaciones estratégicas.
* 📌 **`README.md`**: Este documento, que sirve como introducción y guía general del proyecto.
